Problem:
Ask the user for a number. If the number is divisible by 2, display "Even". Otherwise,
display "Odd".
Algorithm
Step 1: Start.
Step 2: Ask the user to enter a number. Store it as number.
Step 3: Check if number MOD 2 equals 0.
Step 4: If yes, display "Even".
Step 5: If no, display "Odd".
Step 6: End.

Problem:
Ask the user for the length and width of a rectangle. Compute the area and display the
result.
Algorithm:
Step 1: Start.
Step 2: Ask the user to enter the length. Store it as length.
Step 3: Ask the user to enter the width. Store it as width.
Step 4: Compute area = length * width.
Step 5: Display area.
Step 6: End.
